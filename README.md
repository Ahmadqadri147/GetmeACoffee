# GetmeACoffee

## Description

GetmeACoffee is a modern web application built with [Next.js](https://nextjs.org) that enables users to support creators through seamless, real-time coffee purchases. It provides a smooth and secure experience for both supporters and creators, featuring user profiles, payment integration, and an intuitive interface.

## Key Features

- **User Authentication:** Secure sign-in and user session management.
- **Creator Profiles:** View and support creators with personalized pages.
- **Real and Fake Payment Integration:** Supports both testing and real payment processing.
- **Dashboard:** Overview of transactions and user activity.
- **Responsive UI:** Clean, mobile-friendly interface with optimized fonts and assets.
- **API Routes:** Comprehensive backend logic for authentication, payments, profile management, and search.
- **Optimized Performance:** Utilizes Next.js features such as server-side rendering and automatic font optimization.

## Technologies Used

- **Framework:** Next.js (React-based framework for server-rendered applications)
- **Styling:** Tailwind CSS
- **Authentication:** NextAuth.js (custom API routes)
- **Database:** MongoDB (via models/User.js and models/payment.js)
- **Payment Strategies:** Custom payment service with fake and real payment strategies
- **Linting & Formatting:** ESLint, PostCSS
- **Build Tools:** npm / yarn / pnpm / bun
